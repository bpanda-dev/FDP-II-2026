Overview

Most workshops cover many tools. We focus on one complete pipeline, built one stage at a time. By the end of our 5 days, you'll have generated molecules, docked them, and ranked the results.

We will put emphasis on open-source tools for each task, so its accessible. Each session is a short introduction followed by a hands-on lab. Our example uses EGFR in lung cancer, but you can work on your own target if you have an idea to explore.

Mode A short lecture plus a hands-on lab, every session.
Structure 15 sessions of two hours, all building one capstone.
Instructors Bikram Panda and Prof. Debnath Pal. Hands-on: Push Lochan.
Capstone An AI-native Target to Hit pipeline for demonstrating the state of the art in AI-driven research.

The capstone pipeline

Target → Hit

This is the map we keep coming back to. Each stage hands the work to the next. A target name goes in. A ranked hit list comes out.

Stage What you build Tools
Day 1
Target brief		 Literature-mined target dossier + known actives.Frame the target and seed a list of known active compounds. Claude · PubMedBERT
Day 2
Data layer		 AI-coded fetchers: sequence, structure, known actives.Pull clean data from UniProt, the PDB, and ChEMBL. Cursor / Antigravity · Biopython · RDKit
Day 3
Knowledge engine		 RAG + agent target brief (pocket, chemotypes).Give the pipeline a memory that reads the literature and writes a brief. LangChain + ChromaDB · LangGraph
Day 4
Target prep		 Annotated target + defined binding pocket.Characterise the protein and prepare a structure that is ready to dock. ESM-2 · ColabFold
Day 5
Hit finding		 Generated molecules → docked → ranked hit list.Generate candidates, dock them to the pocket, and rank them. REINVENT4 · DiffDock · RDKit

Before the programme

Prerequisites & setup

You'll want a little Python going in. No cheminformatics is assumed. Do the twenty minutes of setup beforehand, and we start on the labs instead of fighting installs.

Prerequisites

Basic understanding of molecular biology and bioinformatics.
Basic Python: variables, loops, functions, installing packages.
Familiarity with Jupyter / Colab notebooks. No prior cheminformatics needed.
A Google account and a Hugging Face account (for model weights).

Pre-workshop requirements

Compute A GPU (incl. NVIDIA H200) so ESM-2, DiffDock and ColabFold run live; cached outputs as a time backup.
LLM access Claude API key (Pro plan); Gemini free tier as a no-cost fallback.
AI IDE Cursor or Google Antigravity (free tiers are enough; both drive frontier models for you).
Model access Hugging Face account for ESM-2 weights.
Python env Python 3.10+, conda/venv; torch, transformers, fair-esm, langchain, langgraph, chromadb, biopython, rdkit; REINVENT4; DiffDock deps.
Data / APIs UniProt, RCSB PDB, ChEMBL, NCBI E-utilities; a shared target dataset folder.

Capstone & assessment

One pipeline, built cumulatively

The capstone runs all week. You build the Target to Hit pipeline for a target you pick, one stage a day. At the end you have a working pipeline, a ranked hit list with scores, and a short demo.

Attendance Presence across the five days.
Daily labs Each session delivers its pipeline stage (Stages 1-5).
Capstone Integrated pipeline + ranked hits + presentation.
Certificate Issued on attendance and capstone completion.

More than any single tool, we hope the shape of the whole thing sticks. Then you can take it back to your own students.
