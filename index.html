Overview

Most workshops tour a dozen tools and hope a few stick. We do the opposite. Everyone builds one pipeline, one stage a day. Every new idea has somewhere to land. By Friday your team has made molecules, docked them, and ranked them.

We pick one tool per job, the best free one. The whole room runs the same stack. Each session is a short talk, then a lab. Our worked example is EGFR in lung cancer. Bring your own target if you would rather.

The capstone pipeline

Target → Hit

This is the map we keep coming back to. Each stage hands the work to the next. A target name goes in. A ranked hit list comes out.

StageWhat you buildTools
Day 1
Target brief		 Literature-mined target dossier + known actives. Frame the target and seed a list of known active compounds. Claude · PubMedBERT
Day 2
Data layer		 AI-coded fetchers: sequence, structure, known actives. Pull clean data from UniProt, the PDB, and ChEMBL. Cursor / Antigravity · Biopython · RDKit
Day 3
Knowledge engine		 RAG + agent target brief (pocket, chemotypes). Give the pipeline a memory that reads the literature and writes a brief. LangChain + ChromaDB · LangGraph
Day 4
Target prep		 Annotated target + defined binding pocket. Characterise the protein and prepare a structure that is ready to dock. ESM-2 · ColabFold
Day 5
Hit finding		 Generated molecules → docked → ranked hit list. Generate candidates, dock them to the pocket, and rank them. REINVENT4 · DiffDock · RDKit

Before the programme

Prerequisites & setup

You'll want a little Python going in. No cheminformatics is assumed. Do the twenty minutes of setup beforehand, and we start on the labs instead of fighting installs.

Prerequisites

Basic understanding of molecular biology and bioinformatics.
Basic Python: variables, loops, functions, installing packages.
Familiarity with Jupyter / Colab notebooks. No prior cheminformatics needed.
A Google account and a Hugging Face account (for model weights).

Pre-workshop requirements

Compute
A GPU (incl. NVIDIA H200) so ESM-2, DiffDock and ColabFold run live; cached outputs as a time backup.
LLM access
Claude API key (Pro plan); Gemini free tier as a no-cost fallback.
AI IDE
Cursor or Google Antigravity (free tiers are enough; both drive frontier models for you).
Model access
Hugging Face account for ESM-2 weights.
Python env
Python 3.10+, conda/venv; torch, transformers, fair-esm, langchain, langgraph, chromadb, biopython, rdkit; REINVENT4; DiffDock deps.
Data / APIs
UniProt, RCSB PDB, ChEMBL, NCBI E-utilities; a shared target dataset folder.

Capstone & assessment

One pipeline, built cumulatively

The capstone runs all week. You build the Target to Hit pipeline for a target you pick, one stage a day. At the end you have a working pipeline, a ranked hit list with scores, and a short demo.
